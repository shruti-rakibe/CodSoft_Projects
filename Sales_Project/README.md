Sales Prediction using Machine Learning

 Project Overview:
This project builds a Machine Learning model to predict sales based on advertising expenditure across different media channels such as TV, 
Radio, and Newspaper.
The goal is to analyze how advertising budgets impact product sales and create a model that can accurately predict future sales.

Dataset:
The dataset contains advertising budgets and corresponding sales figures.

Features:
Column	                    Description
TV	                    Advertising budget spent on TV
Radio	                    Advertising budget spent on Radio
Newspaper                  Advertising budget spent on Newspaper
Sales	                    Product sales (target variable)

How to Run the Project:
Step 1: Install Required Libraries
pip install pandas numpy matplotlib seaborn scikit-learn

Step 2: Place Dataset
Make sure the file sales.csv is in the project folder.

Step 3: Run Python Script
python sales_model.py

Results:
The model predicts sales based on advertising budgets with good accuracy.
Example output:
R2 Score: 0.89
Mean Squared Error: 2.78
