Titanic Survival Prediction

Project Overview:
This project uses Machine Learning to predict whether a passenger survived the Titanic disaster or not. The prediction is based on features such as age, gender, ticket class, fare, and other passenger details.
The Titanic dataset is a popular beginner dataset used to learn data analysis, data preprocessing, and classification algorithms.

Objective:
Build a machine learning model that predicts:
0 → Passenger did not survive
1 → Passenger survived

Dataset Information:
The dataset contains the following features:

Feature	                       Description
PassengerId	 Unique passenger ID
Survived	                      Survival status (0 = No, 1 = Yes)
Pclass	                      Ticket class (1 = First, 2 = Second, 3 = Third)
Name	                      Passenger name
Sex	                      Gender
Age	                      Age of passenger
SibSp	                      Number of siblings/spouses aboard
Parch	                      Number of parents/children aboard
Ticket	                      Ticket number
Fare	                      Ticket price
Cabin	                      Cabin number
Embarked	Port of embarkation

Steps Performed:
1. Import dataset
2. Data cleaning
3. Handle missing values
4. Convert categorical data into numeric form
5. Split dataset into training and testing sets
6. Train machine learning model
7. Evaluate model accuracy
8. Predict survival for new passengers

Machine Learning Algorithm Used:
Logistic Regression

Result:
The Titanic Survival Prediction model was trained using machine learning algorithms and achieved high accuracy. 
Based on features like age, gender, and passenger class, the model successfully predicts whether a passenger survived or not.
